# 🚀 AI Tools Navigator (AI 工具精选导航)

一个现代化的、响应式的 AI 工具集合导航网站。收录了目前市面上最强大的 AI 模型和工具，帮助用户快速找到适合自己的生产力工具。

![Project Screenshot](preview.png)

## ✨ 项目特点

* **全面收录**：覆盖 AI 文本、AI 图像、AI 视频、AI 配音、AI 编程等多个热门领域。
* **精选推荐**：为每个领域筛选出“首选”、“开源”及“平替”方案（如 ChatGPT, DeepSeek, Midjourney, Gemini 等）。
* **极简设计**：深色模式 UI，视觉舒适，专注于内容展示。
* **轻量级**：纯静态页面 (HTML/JS/CSS)，加载速度快，无需复杂后端。

## 🛠️ 技术栈

* **HTML5**
* **CSS3** (Flexbox/Grid 布局)
* **JavaScript** (原生 JS，无需编译)

## 📖 新手保姆级部署教程

本项目非常轻量，**不需要数据库**，**不需要复杂的后端环境**。无论你是想免费部署，还是有自己的服务器，都能在 3 分钟内搞定。

### 方案一：部署到 Cloudflare Pages (推荐，永久免费 & 速度快)
适合：没有服务器、不想花钱、希望网站全球访问速度快的用户。

1.  **注册/登录**：访问 [Cloudflare Dashboard](https://dash.cloudflare.com/) 并注册账号。
2.  **创建项目**：
    * 在左侧菜单点击 **Workers & Pages**。
    * 点击蓝色的 **Create application** 按钮。
    * 点击 **Pages** 标签，选择 **Connect to Git**。
3.  **连接仓库**：
    * 选择你刚才上传的 `ai-tools-nav` 仓库。
    * 点击 **Begin setup**。
4.  **开始部署**：
    * **Project name**：随便填（这会成为你域名的前缀）。
    * **Framework preset**：选 `None` (因为我们是纯静态)。
    * 点击 **Save and Deploy**。
5.  **搞定！** 等待几秒钟，Cloudflare 会给你一个 `xxx.pages.dev` 的免费域名，点击即可访问。

---

### 方案二：部署到宝塔面板 (BT Panel)
适合：自己有云服务器 (VPS) 的用户。

1.  **添加站点**：
    * 登录宝塔面板，点击左侧 **“网站”** -> **“添加站点”**。
    * **域名**：填写你的域名（如 `ai.example.com`）。
    * **PHP版本**：选择 `纯静态` (Static) 或者任意 PHP 版本均可（本项目不需要 PHP，但 Nginx 需要运行）。
2.  **上传文件**：
    * 进入网站的 **根目录** (通常是 `/www/wwwroot/你的域名/`)。
    * 删除目录下自带的 `index.html` 和 `404.html`。
    * 点击 **“上传”**，将本项目的所有文件（`index.html`, `data.js`, `logos文件夹`）上传进去。
    * *(如果你是从 GitHub 下载的 zip 包，记得上传后点击“解压”)*。
3.  **访问**：
    * 在浏览器输入你的域名，即可看到网站。
